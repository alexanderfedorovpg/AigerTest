# AigerTest
Виджет для Aiger Panel 
Установить в деректорию cd /App/
# Воозможные настройки:
<pre> {
        'Title'   - заголовок 
        'Value'   - стандартное значение даты
        'Language' - язык (ru, en)
        'Readonly' - делает недоступным для редактирования пользователем (yes, no)
 }
 </pre>
